Keep Claude Code interactive
while it works.

Approve plans, confirm commands, or choose options from checklists.
Review diffs, tables, and test output before the next step.
Agents can generate custom Ink components and run them inside your terminal.
Avoid losing focus: ask Claude to quiz you, summarize context, or suggest next steps.

Installation

1
Install the CLI and the Claude plugin. 
npm install -g @termosdev/cli
claude plugins install termos
2
Run /termos:start in Claude to start the session.
3
Claude streams interactions with termos up and triggers termos run for questions.
4
Replies wake Claude immediately if it is waiting, or are injected into context when the run finishes.
